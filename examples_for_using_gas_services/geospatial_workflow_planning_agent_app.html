GAS Geospatial Workflow Planning Agent

This Web App is built purely on the Geospatial Workflow Planning Agentic Service. Enter a spatial problem, GAS service URL(s), and your API key. This page streams the planning agent, renders the workflow graph directly in the browser, and provides compact download links for returned artifacts.

GAS Project

Request

This server must expose /agents/geospatial_workflow_planning_agent/tasks.

Enter one GAS server base URL per line. The app automatically builds the GetCapabilities request for each service.

The key is sent only in the ExecuteTask request body and is not saved by this page.

This app requests all planning artifacts as encoded outputs. The graph is rendered client-side from workflow_json.
